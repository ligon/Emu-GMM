* Notable non-obvious decisions worth knowing about

1. *Float64 enabled by default* at =src/emu_gmm/__init__.py=. JAX's
   float32 default is a deep-learning convention; for a statistics
   framework, float32 silently degrades both optimiser convergence
   and recovery accuracy. Documented in commit =74e04b9= and in
   =CLAUDE.md=. Users can override after import.
2. *haliax pinned to git master* in =pyproject.toml=. PyPI haliax
   1.3 imports a removed JAX private API.
3. *Ruff suppresses F722, F821, UP037 project-wide.* These are
   jaxtyping shape-string false positives, not real bugs.
4. *=examples/euler.py= is the canonical demo*. Three acceptance
   tests + =examples/run_euler.py= all depend on it. Hansen-Singleton
   1982 setup; 3 risky assets, M=3, K=2, J_dof=1.
5. *=scipy_lm()= adapter is available but unused in the demo.*
   Before float64, the empirical path required it to certify
   convergence. With float64, =optimistix_lm()= works everywhere.
   The adapter stays in =optimizer.py= for users who want it.
